# Data

The raw data used in this project is **not** included in this repository due to size and licensing considerations. Check the corresponding notebooks to import the data from the official website and to apply the same filters.

## Thera-SAbDab

- Source: Therapeutic Structural Antibody Database (Thera-SAbDab)
- Filters used:
  - Format: whole mAb
  - Highest clinical trial: approved or phase-iii
  - Status: active
  - Genetics: genetically human or humanised

## OAS

- Source: Observed Antibody Space (OAS)
- Filters used:
  - Species: human
  - BSource: PBMC
  - BType: Memory B cells
  - Chain: Heavy
  - Type: IGHG
  - Disease: None
  - Vaccine: None
